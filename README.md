CIFAR-10 CNN with PyTorch
This repository contains a Convolutional Neural Network (CNN) implementation using PyTorch to classify images from the CIFAR-10 dataset.
Project Overview
Dataset: CIFAR-10, 60,000 32x32 color images in 10 classes (50,000 train, 10,000 test)
Goal: Build a CNN to classify CIFAR-10 images into categories like airplane, automobile, bird, cat, deer, dog, frog, horse, ship, truck.
Framework: PyTorch
Key Features:
Convolutional layers
Max Pooling
Fully connected layers
Cross-entropy loss
Adam optimizer
Accuracy evaluation
